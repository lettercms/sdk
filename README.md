LetterCMS is a CMS focused on improving content creation and integrating a built-in way to deliver content via Facebook, Instagram, Twitter, Linkedin, Email, push notifications and more, trying to perform the best user experience, through a image optimizations, lazy load and offline strategies. Deliver the right content to the right users, with a built-in recommendation system based on content filtering and machine learning models to improve conversion rate.

## Official SDK

This monorepo contains the official LetterCMS SDK, to provide access to LetterCMS API on Browsers and NodeJS. and some framework implementations like NextJS
